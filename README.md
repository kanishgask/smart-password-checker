# Smart Password Strength Checker

This project checks the strength of a password and suggests improvements.

## Features

- Detects weak, medium, and strong passwords
- Generates strong password suggestions
- Provides password security tips

## Technologies

- Python
- Regex

## Project Structure

smart-password-checker
│
├── password_checker.py
├── password_generator.py
├── utils.py
├── app.py
├── requirements.txt
└── README.md

## How to Run

python app.py

## Example

Enter a password: hello123

Output:
Password Strength: Medium Password
